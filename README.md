# 狙击外星人（H5）

一个使用原生 HTML、CSS、JavaScript 开发的简单射击练习游戏，无任何框架或游戏引擎依赖。

## 功能概览
- 首页显示最高分与“开始游戏”按钮
- 地图背景、HUD（剩余时间、外星人数量、当前积分）
- 准星替代鼠标，进入游戏区域后准星跟随光标
- 外星人以波次出现（1–3个），5秒内从小变大再变小；未命中则消失
- 计分规则：
  - 命中越快，得分越高（最高约10分，最低1分）
  - 误点（未点中外星人）扣1分
- 过关后可选择继续下一关或返回首页；失败返回首页并保存最高分

## 文件结构
- `index.html`：首页（标题、最高分、开始按钮）
- `game.html`：游戏页（HUD、游戏区域、准星、返回主页按钮）
- `style.css`：样式（背景、外星人、准星、动画）
- `game.js`：逻辑（初始化、外星人波次、计时与计分、事件绑定）

## 素材要求
- 将以下图片放在项目根目录：
  - `level.png`：场景背景（推荐 1920×1080 或更高）
  - `alien.png`：外星人（显示为 30×30px，建议原图 75–150px 以保证清晰）
- 准星采用内嵌 SVG，无需额外文件

## 操作说明
- 进入首页点击“开始游戏”进入游戏页
- 左键点击击中外星人得分；点错扣分
- 右上角“返回主页”按钮可随时返回首页并保存最高分

## 规则细节
- 倒计时：全局 50 秒（`game.js:7`、`game.js:104–113`）
- 外星人：每波 1–3 个，出现位置随机，5 秒脉冲动画（`style.css:86–92`、`game.js:47–77`）
- 计分：命中分数与命中时间成反比（`game.js:191–206`）；误点扣1分（`game.js:208–211`）
- 过关与失败：
  - 清空目标外星人后过关并弹窗（`game.js:124–138`）
  - 时间到失败，返回首页并保存最高分（`game.js:146–152`）

## 常见问题
- 图片不显示：请确认 `level.png`、`alien.png` 位于项目根目录，大小写一致
- npx 无法识别：使用 `npm install -g http-server` 后运行 `http-server`，或改用 Python 方式
- 需要调整难度：可在 `game.js:115–122` 修改目标外星人数量或倒计时；在 `style.css:48–49` 调整外星人显示大小
